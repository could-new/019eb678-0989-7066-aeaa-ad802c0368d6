# 仙侠小说阅读器

基于您提供的精彩仙侠小说片段，构建的跨平台阅读器体验，为您打造出色的移动端和 Web 端小说阅读应用。

## 功能特点
- **章节列表**：清晰的章节选择页面，支持自适应宽屏幕与移动端设备。
- **沉浸阅读**：仿纸质背景色，舒适的行距与字号设计，提供优良的阅读体验。
- **跨平台支持**：全面适配 Web、iOS、Android 及桌面端环境。

## 运行说明
您可以通过以下命令在本地运行体验：
```bash
flutter run
```
